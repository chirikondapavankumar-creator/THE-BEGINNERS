# AI Code Refiner

## Description
AI-powered Java code refiner using Groq LLM.

## Features
- Refactors messy Java code
- Improves structure and readability
- Provides optimization suggestions

## Tech Stack
- Java (Backend)
- Groq API
- Streamlit (Frontend)

## How to Run

### Start Backend
javac backend/*.java  
java backend.Server

### Start Frontend
streamlit run frontend/app.py
